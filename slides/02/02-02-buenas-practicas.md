# Estructura de la aplicación
-------------------------

si abrimos una aplicación Symfony 2 recien instalada, nos encontraremos con la siguiente estructura de archivos.

    Symfony2/
    ├─ app/
    │  ├─ console
    │  ├─ cache/
    │  ├─ config/
    │  ├─ logs/
    │  └─ Resources/
    ├─ src/
    │  └─ AppBundle/
    ├─ vendor/
    └─ web/
